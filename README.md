# CodeAlpha_Amazon_Reviews_Analysis
# Amazon Reviews Analysis

## CodeAlpha Data Analytics Internship

### Project Overview

This project analyzes Amazon customer reviews using Exploratory Data Analysis (EDA), Data Visualization, and Sentiment Analysis.

### Dataset Information

* Total Records: 28,332
* Total Columns: 24

### Tasks Completed

1. Exploratory Data Analysis (EDA)
2. Data Visualization
3. Sentiment Analysis

### Tools Used

* Python
* Google Colab
* Pandas
* Matplotlib
* Seaborn
* TextBlob

### Key Findings

* Positive Reviews: 23,805 (84.0%)
* Neutral Reviews: 2,402 (8.5%)
* Negative Reviews: 2,125 (7.5%)

### Insights

The analysis revealed that the majority of customers expressed positive sentiments toward the products. Product ratings were generally high, indicating strong customer satisfaction.
